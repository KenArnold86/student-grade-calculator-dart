# Student Grade Calculator

This is a Flutter mobile application developed using Dart for the Android App Development course.

## Features

- Enter student name
- Enter course name
- Input assignment score
- Input test score
- Input exam score
- Calculate final score
- Display grade and remark

## Grading System

Assignment: 20%
Test: 30%
Exam: 50%

Grade Scale:
A: 80 – 100
B: 70 – 79
C: 60 – 69
D: 50 – 59
F: Below 50
